### dplyr을 이용한 도수분포표 작성

```r
librarydplyr)
library(tidyverse)

df %>%
    group_by(species, sex) %>%
    count() %>%
    pivot_wider(names_from = sex, values_from = n,
                value_fill = 0) %>% # NA -> 0
    mutate(freq_male = male/(female+male))
```

pivot\_wider 시 values\_from값이 00인 경우 NA로 설정, NA를 특정 값으로 변경하기 위해 `value_fill = 0` 옵션을 사용하여 지정할 수 있다.
